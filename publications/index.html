Neurosymbolic AI: Fundamental Learning for Context Engineering in Computational Linguistics Abstract Large language models make it possible to express symbolic structure directly in natural language, but most context engineering methods assume that humans or external pipelines already know what context the model needs. This work introduces fundamental learning, a two-stage framework where lower-level analyses are generated and reused as auxiliary context to improve higher-level pragmatic prediction.

Rui Mao, Zihao Huang, Xulang Zhang, Erik Cambria

From 10-K to Industry Multi-Labels: A Bottom-Up LLM-Driven Product Clustering with Pareto Frontier Analysis Abstract This work studies industry multi-labeling from firms' 10-K filings using a bottom-up LLM-driven product clustering pipeline. It frames product-level textual evidence as a way to derive industry labels and uses Pareto frontier analysis to reason about the trade-off between coverage and label quality.

Zihao Huang, Joel Quek, Ke-Wei Huang
